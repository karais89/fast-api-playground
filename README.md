# fast-api-playground

환경
- Windows 10
- Pycharm 2020.3
- MySQL
- 공식 튜토리얼의 경우 별도의 브랜치로 연습
- https://fastapi.tiangolo.com/

## 세팅

```sh
pip install fastapi
pip install uvicorn[standard]
```

pycharm 세팅은 생략
