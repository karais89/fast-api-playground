# Tutorial - User Guide - Intro
- https://fastapi.tiangolo.com/tutorial/

이 튜토리얼은 대부분의 기능으로 **FastAPI**를 사용하는 방법을 보여줍니다.

각 섹션은 점차적으로 이전의 것들을 빌드하지만 주제를 구조화하므로 특정 API 요구 사항을 해결하기 위해 특정 항목으로 직접 이동할 수 있습니다.

또한 미래의 참조로 일하기 위해 제작됩니다.

그래서 당신은 당신이 필요로 하는 것을 정확히 볼 수 있습니다.

## 코드를 실행하십시오

모든 코드 블록을 복사하여 직접 복사하여 실제로 테스트 한 Python 파일을 테스트합니다.

예제 중 하나를 실행하려면 코드를 `main.py`로 복사하고 `uvicorn`을 시작하십시오.

```shell
uvicorn main:app --reload
```

코드를 작성하거나 복사하거나 편집하고 로컬로 실행하는 것이 매우 권장됩니다.

편집자에서 사용하는 것이 실제로 FASTAPI의 이점을 보여 주며, 모든 유형의 검사, 자동 완성 등을 작성 해야하는 작은 코드를 보는 것입니다.

## FastAPI 설치

첫 번째 단계는 FASTAPI를 설치하는 것입니다.

튜토리얼의 경우 모든 선택적 종속성 및 기능으로 설치하려고 할 수 있습니다.

```shell
pip install fastapi[all]
```

... 또한 코드를 실행하는 서버로 사용할 수있는 `uvicorn` 포함되어 있습니다.

### Note

파트별로 설치할 수도 있습니다.

응용 프로그램을 프로덕션에 배포하려는 일단 작업을 수행 할 것입니다.

```shell
pip install fastapi
```

또한 `uvicorn` 서버로 작업하십시오.

```shell
pip install uvicorn[standard]
```

사용하려는 선택적 종속성 각각에 대해 동일합니다.

## 고급 사용자 가이드

또한 이 자습서 - 사용 설명서 이후에 나중에 읽을 수 있는 고급 사용자 가이드도 있습니다.

고급 사용자 가이드는 이를 빌드하고 동일한 개념을 사용하고 추가 기능을 가르칩니다.

그러나 먼저 튜토리얼 - 사용 설명서 (현재 읽고있는 것)를 먼저 읽어야합니다.

튜토리얼 사용자 가이드와 함께 완벽한 응용 프로그램을 구축 한 다음 고급 사용자 가이드의 추가 아이디어 중 일부를 사용하여 필요에 따라 다른 방식으로 확장 할 수 있도록 설계되었습니다.
